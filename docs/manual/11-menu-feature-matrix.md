# 11. 메뉴 · 기능 · 시나리오 정리 매트릭스 (2026-05-29)

> 박경수님 — 메뉴 구조가 정돈 안 된 느낌. 모든 메뉴를 **필요기능 / 세부기능 / 사용 시나리오** 3축으로 한눈에 정리.
> 정리 후 사이드바 재배치 또는 메뉴 통합 결정에 활용.

## 박경수님 검토 — 다음 결정 항목

### A. 사이드바 노출 조정
현재 사이드바에 있지만 박경수님 사용 빈도 낮으면 숨기거나, 사이드바에 없지만 박경수님 자주 쓰면 추가.

| 현재 사이드바 | 사용 빈도 (추정) |
|---|---|
| 거래처 | 높음 |
| 전문가 | 높음 |
| 신청 관리 | 높음 |
| 모집 공고 | 중 |
| AI 초안 | 중 |
| 외주·급여 | 높음 |
| 재무 대시보드 | 중 |
| 팀원 관리 | 낮음 |

| 사이드바에 없음 + 자주 쓸 메뉴 | 추가 검토 |
|---|---|
| 포털 관리 (`/portals`) | ⭐ STEP-PORTAL-MULTI-ROLE 후 자주 쓸 예정 |
| 출석 관리 (`/attendance`) | 운영 단계 자주 |
| 일지 통합 (`/activity-logs`) | 운영 단계 |
| 폼 관리 (`/forms`) | 만족도 설문 자주 |
| 견적 (`/estimates`) | 계약 단계 |
| 정산 (`/settlements`) | 종료 단계 |
| 지출 (`/expense`) | 매일 |
| 증빙 (`/receipts`) | 매주 |

### B. 메뉴 통합·축소 후보

| 통합 후보 | 효과 |
|---|---|
| 외주·급여 + 지출 + 증빙 → "지출·증빙" 1개 | 재무 흐름 단순화 |
| 신청 관리 + 모집 공고 + 폼 관리 → "외부 폼" 1개 | 외부 폼 통합 |
| 출석 + 일지 + 수료증 → "운영 기록" 1개 | 운영 흐름 통합 |
| 정산 + 회계 검토 → "정산 흐름" 1개 | 정산 통합 |

### C. 시나리오 기반 사이드바 재구성 (옵션)

```
🏠 홈                대시보드 + 달력

▶ 사업 수주 (Phase 1)
   프로젝트 / 컨소시엄 / 수입·계약 / 거래처

▶ 사업 준비 (Phase 2)
   프로그램 / 전문가 / 외부 폼 [통합]

▶ 사업 운영 (Phase 3)
   강사 포털 / 포털 관리 [신규 추가] / 운영 기록 [통합] / AI 초안

▶ 사업 정산 (Phase 4)
   지출·증빙 [통합] / 정산 흐름 [통합] / 재무 대시보드

⚙️ 시스템
   팀원 관리 / 설정 / 내 페이지
```
